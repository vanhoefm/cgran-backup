Getting usable USRP permissions on Ubuntu
posted by EspeonEefi on Thu 13 of Jul, 2006 [22:44 UTC]
The following makes the USRP /dev files write-accessible by everyone in the usrp group. It is based on scripts extracted from the Debian usrp package.

You want to put the following in /etc/udev/rules.d/60-usrp.rules (highlight and copy and paste to get the entire lines).

# USRP without loaded firmware (FX2)
BUS=="usb", SYSFS{idVendor}=="fffe", SYSFS{idProduct}=="0002", SYSFS{bcdDevice}=="0002", GROUP="usrp", RUN+="/usr/local/bin/usrper load_standard_bits"
BUS=="usb", SYSFS{idVendor}=="fffe", SYSFS{idProduct}=="0002", SYSFS{bcdDevice}=="0004", GROUP="usrp", RUN+="/usr/local/bin/usrper load_standard_bits"

# USRP with loaded firmware
BUS=="usb", SYSFS{idVendor}=="fffe", SYSFS{idProduct}=="0002", SYSFS{bcdDevice}=="0102", GROUP="usrp"
BUS=="usb", SYSFS{idVendor}=="fffe", SYSFS{idProduct}=="0002", SYSFS{bcdDevice}=="0104", GROUP="usrp"

Just for reference, here are those same lines again but allowed to be wrapped.

# USRP without loaded firmware (FX2)
BUS=="usb", SYSFS{idVendor}=="fffe", SYSFS{idProduct}=="0002", SYSFS{bcdDevice}=="0002", GROUP="usrp", RUN+="/usr/local/bin/usrper load_standard_bits"
BUS=="usb", SYSFS{idVendor}=="fffe", SYSFS{idProduct}=="0002", SYSFS{bcdDevice}=="0004", GROUP="usrp", RUN+="/usr/local/bin/usrper load_standard_bits"

# USRP with loaded firmware
BUS=="usb", SYSFS{idVendor}=="fffe", SYSFS{idProduct}=="0002", SYSFS{bcdDevice}=="0102", GROUP="usrp"
BUS=="usb", SYSFS{idVendor}=="fffe", SYSFS{idProduct}=="0002", SYSFS{bcdDevice}=="0104", GROUP="usrp"

(These rules were extracted from the 0.12-1 version of the usrp Debian package. If you want the newest rules, go to the Debian usrp package page, click the "i386" link under "Download usrp," select a mirror, save the .deb file somewhere, do "dpkg -x /path/to/usrp_x.xx-x_i386.deb /path/to/temp/usrp-folder", and then grab the file usrp.rules from /path/to/temp/usrp-folder/etc/udev. I also had to insert "local/" into "/usr/bin/usrper" in order to make this work, since the way Grace (?) installed USRP puts that binary in /usr/local.)

Next, add the group usrp.

sudo addgroup --system usrp

Also, add any users you want to allow to access the USRP to the usrp group. For each user, go

sudo adduser <username> usrp

(Users will have to restart their sessions for the group changes to take effect.)

Finally, unplug and replug-in the radio. Note that now the appropriate device under /dev/bus/usb/*/* has group owner usrp.
